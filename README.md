# Mobile App Assignment

## 📱 Overview
This repository contains Android applications developed as part of the Mobile App Development assignment.

---

##  Q1 - Currency Converter App
- Converts between INR, USD, EUR, JPY
- Uses Spinner for currency selection
- Displays converted value instantly
- Includes Light/Dark theme toggle

---

##  Q2 - Media Player App
- Plays audio file from device (local storage)
- Streams video from URL
- Provides controls:
  - Play
  - Pause
  - Stop
  - Restart

---

## Q3 - Sensor App
- Displays real-time sensor data:
  - Accelerometer (X, Y, Z values)
  - Light Sensor
  - Proximity Sensor
- Uses SensorManager and SensorEventListener

---

## 📁 Project Structure
- `CurrencyConverterApp` → Q1
- `MediaPlayerApp` → Q2
- `SensorApp` → Q3

---

## 🛠️ Technologies Used
- Java
- Android Studio
- XML (UI Design)
